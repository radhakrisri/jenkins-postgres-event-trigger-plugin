Define custom metadata for this job as key-value pairs. This metadata will be recorded in the Supabase jobs table for easier tracking and organization.

Key: A descriptive name for the metadata field (e.g., team, environment, owner, service_type)
Value: Can be either simple text or a JSON object/array

Examples:
  • team: DevOps Team
  • owner: john.doe@company.com
  • environment: production
  • tags: ["critical", "customer-facing"]
  • config: {"region": "us-east-1", "tier": "premium"}
All metadata will be combined into a single JSON object and stored in the jobs table.
